Quem sou eu?

Olá, sou o Daniel Lima tenho 23 anos e trabalho com desenvolvimento c# Desde 2016 onde comecei como Técnico em informática atuando grande parte do tempo como desenvolvedor em um sistema de automação comercial usando .Net windows forms e banco de dados SQL Server, eu atuava no desenvolvimento e no banco de dados desse sistema, assim eu comecei a progredir e me apaixonar por esse universo dev.  

Com o tempo, passei por outras empresas como Desenvolvedor .NET e tive contato com várias outras tecnologias onde trabalhei com .NET Framework, .NET Core 5, Web Api, ASP.NET MVC 5, Web Forms, .NET, Windows Forms, Crystal Reports, Arquitetura em camadas, Padrão DDD, Relatórios usando RDLs (legado),SQL Server, MySql, Preparação do ambiente Azure para Integração de relatórios do Power Bi com o projeto em ASP.NET,Modelagem do banco de dados, Levantamento de Requisitos, Desenvolvimento Front-End (Html, Css, Jquery, Bootstrap) em projetos ASP.NET, Configuração e gerenciamento do Servidor de hospedagem, IIS, Projetos com FIGMA e Adobe XD.

Resumindo um pouco mais de 5 anos de carreira, foram esses os meus envolvimentos com tecnologias, e no meu tempo livre, eu estudo e pratico alguns projetos e deixo salvo em um repositório no GitHub para outros Devs usufruir dos projetos, Sou apaixonado por esse mundo Dev, e gosto muito de um novo desafio.
